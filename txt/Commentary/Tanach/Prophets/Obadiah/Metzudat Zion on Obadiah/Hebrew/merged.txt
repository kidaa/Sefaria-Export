<b>שוכני .</b> היו''ד יתירה : 
<b>בחגוי .</b> ענין בקוע וחריץ כמו יונתי בחגוי הסלע ( ש''ה ב ) :


Verse 4

<b>קנך .</b> מל' קן והוא מדור העופות :


Verse 5

<b>שודדי .</b> ענין גזל ועושק : 
<b>נדמיתה .</b> ענין כריתה ואבדון כמו ער מואב נדמה ( ישעיה טו ) : 
<b>דים .</b> די הספוק כמו והמלאכה היתה דים ( שמות לו ) : 
<b>בוצרים .</b> כן יקראו תולשי ענבי כרם כמו כי תבצור כרמך ( דברי' כד ) : 
<b>עוללות .</b> הם הענבים הקטנים הגרועים כמו לא תעולל ( , שם ) :


Verse 6

<b>נחפשו .</b> מלשון חפוש וחקירה כמו משם אחפש ( עמוס ט ) : 
<b>נבעו .</b> ענין דרישה וחפוש ודומה לו אם תבעיון בעיו ( ישעיה כא ) שהוא ענין דרישת שאלה וכמו שמלת בקשה הונח לדריש' שאלה ולחפוש כן מלת בעה הונח לשניהם : 
<b>מצפוניו .</b> מל' צפון ונסתר :


Verse 7

<b>שלחוך .</b> ענין לויה כמו ואברה' הולך עמם לשלחם ( בראשי' יח ) : 
<b>השיאך .</b> ענין הסתה : 
<b>יכלו .</b> מל' יכולת ונצחון : 
<b>לחמך .</b> ר''ל אוכלי לחמך : 
<b>מזור .</b> ענין מכה ותקרא כן ע''ש שזורים עליה אבק מסממנים המרפאים וכן ולא יגהה מכם מזור ( הושע ה ) : 
<b>תחתיך .</b> במקומך :
