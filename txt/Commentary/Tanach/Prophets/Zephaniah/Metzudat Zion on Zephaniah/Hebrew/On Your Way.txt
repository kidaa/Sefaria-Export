<b>הכהנים .</b> הם מין אחר מתחלפים בלבושיהם :


Verse 5



Verse 6

<b>הנסוגים .</b> ענין החזרה לאחור כמו ולא נסוג ממך ( תהלים פ ) :


Verse 7

<b>הס .</b> ענין שתיקה כמו ויהס כלב ( במדבר יג ) : 
<b>הקדיש .</b> ענין הזמנה כמו קדשו קהל ( יואל ב ) : 
<b>קרואיו .</b> המזומנים לסעודה וכן בראש הקרואים ( ש''א ט ) :

Verse 9

<b>הדולג .</b> ענין קפיצה : 
<b>המפתן .</b> אסקופה כמו מתחת מפתן הבית ( יחזקאל מז ) : 
<b>חמס .</b> עושק :


Verse 10

<b>המשנה .</b> מל' שנים ור''ל בין שתי החומות שהיא משנה לעיר וכן והיא יושבת בירושלים במשנה ( מ''ב כב ) :


Verse 11

<b>הילילו .</b> מל' יללה : 
<b>המכתש .</b> שם מקום בירושלים עמוקה ככלי המכתשת שדוכין בה בשמים : 
<b>נדמה .</b> ענין כריתה כמו ער מואב נדמה ( ישעיה טו ) : 
<b>עם כנען .</b> ר''ל עדת התגרים כמו כנען בידו מאזני מרמה ( הושע יב ) : 
<b>נטילי .</b> ענין משא כמו ונטל החול ( משלי כו ) :


Verse 12

<b>אחפש .</b> ענין בקור ודרישה כמו תמנו חפש מחפש ( תהלים סד ) : 
<b>הקופאים .</b> הצפים ממעל כי ויצף הברזל ( מ''ב ו ) תרגומו וקפא פרזלא : 
<b>שמריהם .</b> הוא עכירת המשקה הנופל לשולי הכלי כמו ושקט הוא אל שמריו ( ירמיה מח ) : 
<b>ירע .</b> מלשון רעה :


Verse 13

<b>חילם .</b> עשרם כמו ישא את חיל דמשק ( ישעיה ח ) : 
<b>למשסה .</b> ענין רמיסה כמו למשסה ואין אומר השב ( שם מב ) :


Verse 14

<b>צורח .</b> ענין שאגת קול גדול כמו יריע אף יצריח ( , שם ) :


Verse 15

<b>עברה .</b> ענין כעס : 
<b>ומצוקה .</b> עניינו כמו צרה וכן בארץ צרה וצוקה ( שם ל ) וכפל המלה בשמות נרדפים להפלגות הענין : 
<b>שואה ומשואה .</b> ענין חושך ושממון כמו אמש שואה ומשואה ( איוב ל ) : 
<b>וערפל .</b> הוא עב הענן :


Verse 16

<b>הפנות .</b> הם המגדלים שדרכם לעמוד בפנות וזויות המבצר :


Verse 17

<b>והצרותי .</b> מל' צרה : 
<b>ולחומם .</b> כן יקרא גם הבשר כמו כי לחמם לנפשם ( הושע ט ) : 
<b>כגללים .</b> כצואה כמו כגללו לנצח יאבד ( איוב כ ) :


Verse 18

<b>קנאתו .</b> ענין כעס וחמה כמו הם קנאוני בלא אל ( דברים לב ) : 
<b>תאכל .</b> ענין השחתה : 
<b>נבהלה .</b> מל' בהלה :

<b>בקשו .</b> ענין דרישה :


Verse 4

<b>ואפסי .</b> כמו ואין : 
<b>ישרוק .</b> ענין השמעת קול הבאה בקבוץ השפתים כמו שרקו ויניעו ראשם ( איכה ב' ) : 
<b>יניע .</b> מלשון הנעה והנדה :



Chapter 3



Verse 1

<b>מוראה ונגאלה .</b> ענין שניהם מלוכלך ומטונף כמו ושמתיך כרואי ( נחום ג ) וכמו נגאלו בדם ( ישעיה נט ) וכפל המלה בשמות נרדפות : 
<b>היונה .</b> מל' אונאה ועניינו קחת ההון וכן מפני חרון היונה ( ירמיה כה ) :


Verse 2



Verse 3

<b>גרמו .</b> ענין עצם כמו גרמיו כמטיל ברזל ( איוב מ ) :


Verse 4

<b>פוחזים .</b> קלי הדעת וכן אנשים ריקים ופוחזים ( שופטים ט ) : 
<b>בוגדות .</b> מל' בוגד : 
<b>חללו .</b> ענין בזיון : 
<b>חמסו .</b> ענין הסרה כמו ויחמוס כגן שכו ( איכה ב ) :


Verse 5

<b>לאור .</b> ר''ל ברור כאור ודומה לו : 
<b>נעדר .</b> ענין חסרון כמו איש לא נעדר ( ישעיה מ ) :


Verse 6

<b>נשמו .</b> מל' שממון : 
<b>פנותם .</b> הם המגדלים הגבוהים כמו ועל הפנות הגבוהות ( לעיל א ) : 
<b>החרבתי .</b> מל' חרבן : 
<b>חוצותם .</b> ענין שוק ורחוב : 
<b>נצדו .</b> ענין שממון כי והשמותי אני את הארץ ( ויקרא כו ) ת''א ואצדי :


Verse 7

<b>אך .</b> עניינו כמו באמת וכן אך עצמי ובשרי אתה ( בראשית כט ) : 
<b>מעונה .</b> מדורה כמו ה' מעון אתה היית ( תהלים צ ) : 
<b>פקדתי .</b> צויתי כמו והוא פקד עלי ( עזרא א ) : 
<b>עלילותם .</b> מעשיהם :


Verse 8

<b>חכו .</b> ענין המתנה ואיחור כמו ונסתה ולא תחכה ( מ''ב ט ) : 
<b>לעד .</b> ענין בזה ושלל כמו בבוקר יאכל עד ( בראשית מט ) :


Verse 9

<b>שפה .</b> כן יקרא לשון עם ועם וכן שפת רעהו ( שם יא ) : 
<b>ברורה .</b> כלפי שקורא לשון העכו''ם עמקי שפה כמ''ש עם עמקי שפה ( ישעיה לג ) קורא למולם לשון הקדש שפה ברורה ונקיה : 
<b>שכם .</b> ר''ל צד אחד ודעה אחת כמו דרך ירצחו שכמה ( הושע ו ) :


Verse 10

<b>עתרי .</b> שם אומה : 
<b>בת פוצי .</b> עדת פוצי והוא שם אומה : 
<b>יובילון .</b> ענין הבאה כמו יובל שי ( ישעיה יח ) :


Verse 11

<b>עלילותיך .</b> מעשיך : 
<b>עליזי .</b> ענין שמחה :


Verse 12

<b>עני ודל .</b> ר''ל מוכנע כעני ודל : 
<b>וחסו .</b> ענין בטחון :


Verse 13

<b>תרמית .</b> מל' רמיה ומרמה : 
<b>ירעו .</b> מל' מרעה : 
<b>ורבצו .</b> ענין השכיבה לנוח :


Verse 14



Verse 15

Verse 16

Verse 17

<b>יחריש .</b> ענין שתיקה כמו והחריש לה ( במדבר ל ) :


Verse 18

<b>נוגי .</b> מל' יגון ותוגה : 
<b>ממועד .</b> ענין זמן ועת כמו למועד אשוב אליך ( בראשית יח ) : 
<b>אספתי .</b> ענין הכנסה כמו ואין איש מאסף אותם ( שופטים יט ) : 
<b>משא .</b> מל' משא וסבל :


Verse 19

<b>עושה .</b> ענין כתישה ומעוך כמו בעשות ממצרים דדיך ( יחזקאל כג ) : 
<b>מעניך .</b> מל' עינוי : 
<b>הצולעה .</b> החגרת כמו והוא צולע על ירכו ( בראשית לב ) : 
<b>והנדחה .</b> מלשון דחיה : 
<b>לתהלה .</b> מל' הלול ושבח : 
<b>לשם .</b> ר''ל לפרסום שם טוב :


Verse 20

<b>בשובי .</b> ענין השקט ומרגוע כמו בשובה ונחת ( ישעיה ל ) :
