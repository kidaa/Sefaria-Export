<b>ויך .</b> יאמר כן בדרך השאלה על התפשטות ודוגמתו ומחה על כתף ים כנרת ( במדבר לד ) ומחה הוא תרגום של מכה : 
<b>כלבנון .</b> שם יער :


Verse 7

<b>יונקותיו .</b> ענפיו הרכים וכן ויעל כיונק לפניו ( ישעיה נג ) : 
<b>הודו .</b> ענין הדר ונאה :


Verse 8

<b>ישובו .</b> ענין ההשקט ומרגוע כמו בשובה ונחת ( שם ל ) : 
<b>יחיו .</b> ענין השבת הנפש מן החולשה ודומה לו עד חיותם ( יהושע ה ) : 
<b>דגן .</b> כדגן ותחסר כ''ף הדמיון וכן ייטיב גהה ( משלי יז ) ומשפטו כגהה :


Verse 9

<b>לעצבים .</b> הם העכו''ם : 
<b>עניתי .</b> מלשון עניים ותשובת שאלה : 
<b>ואשורנו .</b> ענין הבטה וראיה כמו אשורנו ולא קרוב ( במדבר כד ) : 
<b>כברוש .</b> שם אילן : 
<b>רענן .</b> ענין לחות ורטיבות כמו בשמן רענן ( תהלים צב ) :
